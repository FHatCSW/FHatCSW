Selected Highlights

Trustpoint – Open Source Trust Anchor

Co-created and technically led the development of Trustpoint, an open-source trust anchor and PKI platform developed together with contributors from multiple organizations. Successfully completed a Cyber Resilience Act (CRA) assessment for the project.

CMP Server Implementation (RFC 9483)

Implemented a complete server-side implementation of the Certificate Management Protocol (CMP) according to RFC 9483 in Python—including extensive ASN.1 work that was as educational as it was challenging. (ASN.1 nearly drove me crazy, but we got there!)

Secure Development Lifecycle

Established and standardized modern engineering practices including type checking, linting, automated testing, static code analysis (CodeQL), dependency management, SBOM generation (CycloneDX, SPDX), OWASP ZAP integration, and OpenSSF badges—automating what can be automated to improve software quality and security across the project.

Industrial Connectivity & IoT Solutions

Implemented secure industrial communication solutions using OPC UA (including GDS Push implementation), MQTT, Keyfactor's EJBCA and SignServer, alongside IoT stacks including Grafana, Telegraf, InfluxDB, Node-RED (published 2 packages on NPM), Mosquitto, and Mender. Extensive hands-on work with Raspberry Pi and ESP32 platforms.
